Our Offer

JavaScript has become a powerful tool supporting any profitable web business because of its uncountable features and portability. You just can't live without it, if you want to deliever a unique experience to your users on the web — both on desktop and mobile. As we have been living in the JS world for years we exactly know its best parts and how you can benefit from them as a web developer or entrepreneur.

We have spent thousands of hours working on JavaScript projects. Now we want to share our expertise with you during one of our dedicated JavaScript workshops. We don't focus on academic knowledge. We won't overwhelm you with tons of books. Instead, we will teach you real life examples and conduct practical exercises. By signing up to our workshops, you will learn things you can use at your work as a JavaScript developer right away! Learn about our three fundamentals:

Basic JavaScript workshop

You will learn basic programming concepts as well as understand how browsers work. You will also code your first JavaScript application using drag and drop functionality and HTML5 APIs. Moreover, we will go through DOM fundamentals and learn how to manipulate it without support from external libraries.

Workshop Topics

  • Programming intro ~120 min
  • Browser programming intro ~60 min
  • My first app ~180 min
  • What is DOM? ~120 min
  • Animations and interactivity ~60 min
  • HTML5 & CSS vs. JavaScript ~120 min
